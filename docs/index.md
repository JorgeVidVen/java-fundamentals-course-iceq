# Java Fundamentals Course ICEQ

Curso introductorio de Java para estudiantes de secundaria. La ruta está pensada para trabajar desde el navegador, practicar en Replit y terminar con una aplicación de consola llamada **Aventura del Número Secreto**.

## Ruta de aprendizaje

| Nivel | Tema | Producto |
|---|---|---|
| 01 | Primer programa en Java | Mensaje en consola |
| 02 | Variables y tipos de datos | Ficha de estudiante |
| 03 | Condiciones y bucles | Reglas e intentos |
| 04 | Métodos y entrada por consola | Calculadora mínima |
| 05 | Clases y objetos | Modelo de jugador |
| 06 | Encapsulación y proyecto final | Aventura del Número Secreto |

## Dinámica recomendada

```text
Ejecutar -> Cambiar una cosa -> Ejecutar -> Explicar
```

La meta de cada lección es lograr una salida visible, corregir al menos un error y explicar qué cambió el código.

## Proyecto final

En la última lección se integra todo en **Aventura del Número Secreto**:

- entrada por consola;
- vidas e intentos;
- niveles con dificultad;
- puntaje y recompensas;
- clases separadas por responsabilidad;
- encapsulación para proteger el estado del jugador.

Abre la práctica final aquí: [Lección 06: Proyecto final](lessons/lesson-06-final-project/part-02-practice.md).
