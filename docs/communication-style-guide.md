# Glosario técnico del curso

Este glosario resume términos usados durante el curso. El objetivo es mantener un lenguaje claro para estudiantes y para la revisión del material.

## Términos del curso

| Término | Significado en este curso |
|---|---|
| Biblioteca | Conjunto de clases o herramientas disponibles para usar. |
| Requisito | Condición que debe cumplir el curso, la práctica o el programa. |
| Aplicación | Programa que resuelve una tarea para un usuario. |
| Desplegar | Publicar o preparar una solución para uso. |
| Probar | Ejecutar y verificar resultados esperados. |
| Hacer commit | Registrar un cambio en Git cuando el repositorio del equipo esté publicado. |
| Sección, tabla, vista, formulario o barra lateral | Usar el nombre específico del elemento cuando corresponda. |
| Marco de trabajo Scrum | Forma de organizar trabajo en equipo mediante roles, eventos y entregas incrementales. |

## Lectura recomendada

Cuando aparezca una palabra nueva en una lección, vuelve a este glosario y revisa el significado antes de continuar con el código. El objetivo no es memorizar definiciones largas, sino reconocer el término cuando aparezca en ejemplos, errores de Java o instrucciones de práctica.
