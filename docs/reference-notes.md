# Referencias externas

Estas referencias sustentan la precisión técnica del curso. Los enlaces se mantienen visibles para que el material pueda revisarse y ampliarse después.

## Referencias usadas

1. Dev.java: Getting Started with Java
   URL: https://dev.java/learn/getting-started/
   Uso: ciclo escribir, compilar, ejecutar; errores comunes; enfoque moderno.

2. Oracle Java Tutorials: Hello World
   URL: https://docs.oracle.com/javase/tutorial/getStarted/application/index.html
   Uso: clase, método `main` y `System.out.println`.

3. Oracle Java Tutorials: Language Basics
   URL: https://docs.oracle.com/javase/tutorial/java/nutsandbolts/index.html
   Uso: variables, operadores, expresiones, bloques y control de flujo.

4. Oracle Java Tutorials: Classes and Objects
   URL: https://docs.oracle.com/javase/tutorial/java/javaOO/index.html
   Uso: clases, objetos, constructores y métodos.

5. Replit Docs: Welcome and import flows
   URL: https://docs.replit.com/build/welcome
   URL: https://docs.replit.com/build/import-from-providers
   Uso: justificar prácticas desde navegador e integración con GitHub.

6. MOOC.fi Java Programming
   URL: https://java-programming.mooc.fi/part-1/
   Uso: observar progresión real de curso: primeros pasos, impresión, entrada, variables y condiciones.

## Alcance adoptado

- Se usa Replit como herramienta principal por consistencia.
- No se exige instalación de JDK.
- Se limita el alcance: no herencia, interfaces, paquetes, colecciones avanzadas ni pruebas automatizadas.
- Se mantiene el proyecto final como aplicación de consola para que el tiempo total siga cerca de una hora.
