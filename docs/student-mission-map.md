# Mapa de misión del estudiante

Este mapa convierte el curso en una ruta de seis niveles. La meta no es leer todo de una vez, sino completar pequeñas victorias ejecutables en Java.

## Cómo jugar la ruta

1. Abre la lección.
2. Ejecuta el starter antes de editar.
3. Completa la misión principal.
4. Resuelve un error intencional.
5. Marca la insignia de la lección.
6. Avanza al siguiente nivel solo si puedes explicar tu cambio en una frase.

## Ruta de niveles

| Nivel | Nombre de misión | Archivo principal | Victoria visible | Insignia |
|---|---|---|---|---|
| 01 | Despertar Java | `starter-files/lesson-01-hello-world/Main.java` | La consola muestra un mensaje propio. | Primer Run |
| 02 | Inventario de variables | `starter-files/lesson-02-variables/Main.java` | El programa imprime una ficha de estudiante. | Datos bajo control |
| 03 | Puerta de decisiones | `starter-files/lesson-03-if-loops/Main.java` | El programa decide y repite intentos. | Bucle explorador |
| 04 | Taller de métodos | `starter-files/lesson-04-methods/Main.java` | Una calculadora usa métodos pequeños. | Receta reutilizable |
| 05 | Laboratorio de objetos | `starter-files/lesson-05-classes-objects/` | Un jugador conserva nombre y puntaje. | Modelador inicial |
| 06 | Aventura del Número Secreto | `starter-files/lesson-06-final-project/` | El juego final integra clases, entrada, vidas y puntaje. | Guía de Java |

## Puntos de práctica

Usa esta tabla para medir avance sin competir por velocidad.

| Acción | Puntos sugeridos |
|---|---:|
| Ejecutar el starter sin cambios | 5 |
| Completar la misión principal | 20 |
| Corregir un error intencional | 15 |
| Explicar el cambio a un compañero | 10 |
| Resolver un reto de extensión | 20 |

## Reglas de avance

- Si el código no ejecuta, todavía no termina la misión.
- Si la respuesta fue copiada sin entender, repite el paso con una variación pequeña.
- Si el error aparece, lee la primera línea del mensaje antes de borrar código.
- Si el reto principal ya funciona, cambia un dato y vuelve a ejecutar.

## Reto final opcional

Cuando completes la Lección 06, agrega una mejora pequeña al juego:

- un nombre de dificultad;
- un mensaje final distinto;
- una recompensa por ganar un nivel;
- una pista extra;
- un título para el jugador según su puntaje.

La mejora debe poder explicarse con los temas del curso: variables, condiciones, bucles, métodos, clases y encapsulación.
